sh-C Polyglot

A nice (mis)feature of POSIX shells is that the shell must run a script that's marked as executable even if the shebang line is missing.

This does make sense, in case the underlying system has no support for it. More details at http://www.in-ulm.de/~mascheck/various/shebang/

However, all modern *nix systems do support it, of course.

A guy on IRC said:

I'm sure it'd be possible to write a file which would print out the lyrics to some song if executed by csh, search for big prime numbers if compiled as C code, and erase your /usr folder if run as a bash script ;)

Well, I don't know csh but I'm up to the challenge :)

#include <stdio.h>
main () { printf /* # */ ("primes: 2 3 5 7 and so on\n");  } /*
} ; echo some shell code that deletes your files # */

Heh, it wasn't even too hard. :P
